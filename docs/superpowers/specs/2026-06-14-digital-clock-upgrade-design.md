# 2026-06-14 디지털 시계 프로젝트 기능 업그레이드 설계서 (Digital Clock Upgrade Spec)

본 문서는 데스크탑 디지털 탁상시계 PWA 웹 앱의 새로운 기능(폰트 교체, 미세먼지 정보 추가, 수원 영통 지역 추가, OLED 번인 방지, 주야간 자동 테마 변경)을 추가하기 위한 상세 설계 사양을 정의합니다.

---

## 1. 요구사항 정의 (Requirements)

### 폰트 교체 기능 추가
- **기존 상태**: `Chivo Mono` 단일 폰트 고정.
- **개선 사양**: 세로로 긴 시계용 폰트 2종(`Teko`, `DSEG7`)을 추가하여 설정 패널에서 선택 가능하도록 구현.
- **디자인 원칙 (분리형)**: 시계 숫자에만 새로 추가한 스타일 폰트를 적용하고, 한글 가독성 유지를 위해 날짜 및 날씨 텍스트는 시스템 고딕 폰트(Mixed Style)를 유지함.

### 날씨 및 미세먼지 정보 연동
- **기존 상태**: 기온 및 기상 상태만 연동.
- **개선 사양**: [Open-Meteo Air Quality API](https://open-meteo.com)를 연동하여 PM10, PM2.5 미세먼지 정보를 실시간 연동.
- **UI 표기 옵션**: 설정에서 토글로 변경 가능하도록 처리.
  - **심플 배지형 (기본)**: `맑음 · 24°C · [미세 좋음]` 형식 (PM10 기준 등급만 심플 배지로 표기).
  - **상세 수치형**: `맑음 · 24°C | 미세 15(좋음) · 초미세 8(좋음)` 형식 (PM10, PM2.5의 구체적 수치와 등급 표기).

### 수원 영통 지역 추가
- **개선 사양**: 기상 정보 설정 지역 리스트에 "수원 영통" 추가.
- **좌표 사양**: 위도 `37.2593`, 경도 `127.0506`

### OLED 번인 방지 대책
- **개선 사양**: OLED 화면 보호를 위한 미세 픽셀 시프트(Micro Pixel Shift) 로직 구현 및 설정 토글 추가.
- **작동 원리**: 번인 방지 옵션이 켜져 있을 때, 1분 주기로 메인 시계 컨테이너(`clock-display`)를 상하좌우 `-4px ~ +4px` 범위 내에서 임의로 미세하게 이동시킴(`transform: translate(x, y)` 활용). 시인성 저해 없이 픽셀 고착을 예방.

### 주야간 자동 테마 모드 추가
- **개선 사양**: 시간에 맞춰 LED 광원 색상이 자동으로 변하는 모드 추가 및 설정 제공.
- **설정 항목**:
  - 주야간 자동 테마 모드 On/Off 토글
  - 주간 시작 시간 (예: 06:00, 0~23시 선택 가능)
  - 주간 종료/야간 시작 시간 (예: 24:00, 0~23시 선택 가능)
  - 주간 테마 색상 (초기값: 에메랄드 그린)
  - 야간 테마 색상 (초기값: 앰버 오렌지)
- **작동 원리**: 시계 틱 루프에서 현재 시각(Hour)을 감시하여 주간/야간 시간대에 해당하는 테마 컬러를 실시간으로 자동 적용 (자동 모드 활성화 시 수동 테마 선택 버튼은 비활성화).

## 2. UI/UX 및 설정 서랍 변경안

설정 서랍(`index.html` 내 `#settings-panel`)에 아래 항목들을 순차적으로 추가 및 업데이트합니다:

1. **LED 테마 색상** (주야간 모드 켜질 시 비활성화 스타일 처리)
2. **시계 폰트 스타일** [NEW]
   - `<select id="font-select">` 엘리먼트 제공. Chivo Mono, Teko, DSEG7 선택 가능.
3. **주/야간 자동 색상 모드** [NEW]
   - 토글 스위치 및 설정 서브 패널 추가.
   - 주간 시작 시간, 야간 시작 시간, 주간 테마, 야간 테마를 설정할 수 있는 컨트롤 탑재.
4. **미세먼지 상세 수치 표시** [NEW]
   - 토글 스위치 제공. 활성화 시 상세 텍스트(B안), 비활성화 시 심플 배지(A안).
5. **OLED 번인 방지 모드** [NEW]
   - 토글 스위치 제공. 켜짐 시 픽셀 시프트 활성화.

## 3. 기술적 구현 계획

### A. 폰트 리소스 다운로드 및 배치
- `assets/fonts/Teko-Medium.ttf` 및 `assets/fonts/DSEG7Classic-Bold.ttf` (혹은 `.woff2`) 리소스를 프로젝트에 배치합니다.
- `style.css`에 `@font-face` 추가:
  - `Teko`: `font-family: 'Teko'`
  - `DSEG7`: `font-family: 'DSEG7'`

### B. 미세먼지 API 호출 로직
- `app.js` 내 `fetchWeather()` 수정:
  - 기존 기상 정보 호출 시, 동시에 혹은 비동기 병렬로 Air Quality API(`https://air-quality-api.open-meteo.com/v1/air-quality?latitude={lat}&longitude={lon}&current=pm10,pm2_5`)를 호출합니다.
  - 응답 데이터(`current.pm10`, `current.pm2_5`)를 받아 한국 미세먼지 기준 등급을 계산합니다:
    - **PM10 등급**: 0-30 좋음, 31-80 보통, 81-150 나쁨, 151+ 매우 나쁨
    - **PM2.5 등급**: 0-15 좋음, 16-35 보통, 36-75 나쁨, 76+ 매우 나쁨
  - 사용자가 설정한 미세먼지 표기 옵션(`state.showDustDetails`)에 따라 DOM을 구성합니다:
    - 심플 배지형: 미세먼지 등급 하나만 색상 매칭된 배지로 렌더링.
    - 상세 수치형: 미세먼지 및 초미세먼지의 수치와 등급을 한눈에 표기.

### C. OLED 번인 방지 (픽셀 시프트) 구현
- `app.js`에 번인 방지 인터벌 로직 추가:
  - `state.burnInPrevention`이 `true`인 경우, `setInterval`을 통해 1분(60,000ms)마다 `#app-container` 또는 `.clock-display` 엘리먼트에 `transform: translate(x, y)` 인라인 스타일을 랜덤 부여합니다 (x, y 범위: -4px ~ 4px).
  - 옵션을 끄면 `transform` 스타일을 `none`으로 원복합니다.

### D. 주야간 자동 테마 모드
- `app.js` 상태에 `autoTheme`, `dayStart`, `dayEnd`, `dayTheme`, `nightTheme` 추가 및 LocalStorage 연동.
- 매초 작동하는 `updateClock` 루프 혹은 별도의 시간 감시 인터벌에서 현재 시간이 주간/야간 시간대에 해당하는지 판단합니다:
  - 예: 시작=6, 종료=24인 경우 `currentHour >= 6 && currentHour < 24` 이면 주간, 그 외는 야간.
  - 해당 시간대의 테마 색상을 `document.documentElement.setAttribute('data-theme', theme)`로 즉시 반영합니다.

## 4. 검증 계획 (Verification Plan)

### 수동 검증
1. **폰트 변경 테스트**: 설정 창에서 Chivo Mono -> Teko -> DSEG7로 변경 시 시계 숫자의 폰트가 잘 바뀌는지, 레이아웃 깨짐이 없는지 확인.
2. **미세먼지 정보 및 수원 영통 테스트**: 지역을 "수원 영통"으로 설정한 후 날씨 정보와 미세먼지 등급이 정상 노출되는지 확인. 미세먼지 상세 옵션 토글 시 레이아웃이 유연하게 갱신되는지 검증.
3. **번인 방지 테스트**: 번인 방지 모드를 켜고 임시로 주기를 3초 등으로 단축 설정하여 시계 글자가 미세하게 움직이는지 눈으로 검증.
4. **주야간 모드 테스트**: 주야간 모드 활성화 시 현재 시간대에 맞추어 설정한 색상이 적용되는지, 시간 설정을 임의로 조정해 가며 테마가 실시간 자동 변환되는지 검증.
